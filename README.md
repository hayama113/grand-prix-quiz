# Grand Prix Quiz v1.5

合計10,000問版です。

## 問題構成

- 既存問題：1,000問
- F1関連追加：2,000問
- 他モータースポーツ：2,000問
- スーパーカー：2,000問
- 一般車：3,000問

## 配置

ZIPを解凍し、中のファイルをGitHubリポジトリ直下へ配置してください。

## 必須ファイル

- index.html
- style.css
- app.js
- questions.json
- question-data.js
- manifest.json
- service-worker.js

## キャッシュ

v1.5では service-worker のキャッシュ名を `grand-prix-quiz-v1-5` に更新しています。
